# Maison de Saveur API

Backend service for the **Maison de Saveur** recipe sharing application.
Built with Node.js, Express, and Supabase.

## Features
- Secure Authentication (JWT & bcrypt)
- Profile Management
- Recipe Creation & Discovery
- File Uploads via Supabase Storage

## Setup
To run this project locally, make sure you have a `.env` file configured with your Supabase credentials.

```bash
npm install
npm run start
```
